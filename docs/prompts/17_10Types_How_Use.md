## 🚀 문제 해결 프롬프트: 멀티 플랫폼 콘텐츠 변형(Repurposing) 전략 기획

### 👑 프롬프트 본문

> **[전문가 페르소나 및 역할 지정]**
> 당신은 미디어 에이전시의 **'멀티미디어 콘텐츠 변형(Repurposing) 전문가'**이자 **'크로스 플랫폼 배포 전략가'**입니다. 클라이언트인 대형 식품 기업의 신규 '친환경 밀키트' 브랜드 런칭 캠페인의 성공적인 콘텐츠 배포를 위해 다음의 지침을 따르십시오.
>
> **[주어진 원본 소스 및 제약 사항]**
> 1.  **원본 소스:** 15분 분량의 고화질 홍보 영상 및 스크립트.
> 2.  **핵심 내용:** 제품 개발 과정, 재료의 신선함, 조리 편의성.
> 3.  **예산 제약:** 추가 촬영은 불가능하며, 오직 이 15분 원본 소스만을 활용해야 합니다.
>
> **[요청 사항: 4단계 구체적 기획안]**
>
> **1. 핵심 콘텐츠 분할 및 주제 추출:**
> * 15분 원본 영상에서 추출할 수 있는 **최대 5가지의 핵심 메시지/주제**를 명확히 분할하고, 각각에 **플랫폼별 활용을 고려한 매력적인 제목(후킹 포인트)**을 부여하세요. (예: 1~3분: 개발 스토리, 3~5분: 재료 신선도 강조 등)
>
> **2. 플랫폼별 콘텐츠 변형 전략 (Repurposing Plan):**
> * 아래 4개 플랫폼 각각에 대해, 추출된 핵심 메시지를 활용하여 **최적화된 콘텐츠 변형 기획안**을 구체적으로 제시하세요.
>     * **플랫폼 A: YouTube (롱폼/숏츠 혼합)**: 롱폼 유지 구간, **30~60초 분량의 숏츠**로 재가공할 **타임라인 구간(예: 3:00~3:50)**과 해당 숏츠의 **핵심 후킹 문구**를 포함하여 설명하세요.
>     * **플랫폼 B: Instagram (릴스/캐러셀)**: 재료의 '신선함'을 강조할 수 있는 **15초 이내의 릴스** 2개 기획안과, 조리 과정을 압축한 **이미지 캐러셀(5장 구성)** 기획안의 내용을 구체적으로 분할하여 제시하세요.
>     * **플랫폼 C: TikTok:** 밀키트의 '조리 편의성'을 **강하게 강조**하는, 트렌디한 **10초 이내의 숏폼** 기획안 1개를 제시하고, 사용해야 할 **유행 사운드/밈**의 컨셉을 제안하세요.
>     * **플랫폼 D: Naver Blog (텍스트 중심)**: 영상 스크립트와 장면 캡처를 활용하여, 검색 유입을 극대화할 수 있는 **SEO 최적화된 블로그 포스팅 목차(제목 포함)**를 제시하고, '친환경' 키워드를 어떻게 녹일지 설명하세요.
>
> **3. 콘텐츠 제작 시 주안점:**
> * Repurposing 과정에서 **'친환경'**과 **'신선함'**이라는 핵심 브랜드 가치를 일관성 있게 유지하기 위한 **편집/텍스트/디자인 측면의 3가지 주요 지침**을 간결하게 요약하세요.
>
> **4. 실행 결과 요약 테이블:**
> * 요청한 모든 플랫폼별 변형 콘텐츠 유형, 길이, 핵심 주제를 한눈에 볼 수 있도록 요약 **테이블**로 정리하세요.


## 📚 프롬프트 분석 및 개선 방향

### 1. 프롬프트 유형과 선택 이유

| 항목 | 내용 |
| :--- | :--- |
| **프롬프트 유형** | **복합 지시 및 제약 기반 생성 프롬프트 (Constraint-Based & Role-Playing)** |
| **선택 이유** | * **역할 부여 (Role-Playing):** '멀티미디어 콘텐츠 변형 전문가' 페르소나를 지정하여, AI의 응답이 **실무적이고 전문적인 기획안**의 형태를 갖추도록 유도했습니다. |
| | * **구체적인 제약 조건 설정 (Constraint-Based):** '15분 원본 소스만 사용', '추가 촬영 불가' 등 현실적인 제약을 명확히 설정하여, AI가 **제한된 자원을 활용하는 창의적 재가공 능력**을 집중적으로 발휘하도록 강제합니다. |
| | * **단계별/플랫폼별 명확한 지시:** 각 요청 사항(1~4단계)과 플랫폼별 요구사항(길이, 형식, 목표)을 구체적으로 분할 지정하여, AI가 **추상적인 답변** 대신 **실행 가능한 구체적인 전략**을 생성하도록 유도했습니다. |
| | * **결과 포맷 지정 (테이블):** 최종 결과물을 요약 테이블로 정리하도록 지시하여, **가독성과 실무 활용도**를 극대화했습니다. |

### 2. 향후 보다 나은 프롬프트 작성을 위해 추가적으로 필요한 것

현재 프롬프트는 콘텐츠 변형 자체에는 최적화되어 있으나, 실질적인 마케팅 전략 수립의 깊이를 더하기 위해 다음 요소들을 추가할 수 있습니다.

| 추가 필요 요소 | 구체적 추가 지시 예시 | 기대 효과 |
| :--- | :--- | :--- |
| **타겟 오디언스 및 페인 포인트 지정** | "이 밀키트의 주 타겟은 **30대 맞벌이 부부**입니다. 이들의 **시간 부족**이라는 주요 pain point를 각 플랫폼의 후킹 메시지에 반영해 주세요." | 타겟 고객의 니즈에 정확히 부합하는 **맞춤형 마케팅 메시지**를 도출하여 전략의 실효성을 높입니다. |
| **KPI (핵심 성과 지표) 설정** | "유튜브 롱폼은 **'평균 시청 지속 시간'**을 높이고, 인스타그램 릴스는 **'저장 수'**를 극대화하는 방향으로 전략을 구상해 주세요." | 콘텐츠 변형이 단순 기획에서 그치지 않고, **실제 마케팅 목표(KPI)** 달성에 기여하도록 AI의 전략적 사고를 유도합니다. |
| **벤치마킹 및 톤앤매너 요구** | "콘텐츠 톤앤매너는 **'신뢰성 있고, 빠르며, 세련된'** 느낌을 강조해 주세요. 영상 편집 시 **친환경적인 색감(Greenish)을 필수로 적용**하도록 지침을 추가하세요." | 원하는 결과물의 **시각적/감성적 분위기**를 정확히 정의하여, AI가 더 일관성 있고 원하는 브랜드 이미지에 맞는 결과물을 생성하도록 합니다. |
